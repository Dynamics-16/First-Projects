Things I need to improve!

There are definitely many things I need to work on. In due course I'm sure I will get a whole lot better! Right now, here are the main focuses (off the top of my head):


  1. The overall structure of my HTML

    I'm slowly getting used to the structure of HTML - where certain elements should be placed, what should (and shouldn't) be nested inside other elements and the bare requirements for a HTML document.

    The code below is an example that displays the essential elements within a HTML document!

    
            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>My Web Page</title>
            </head>
            <body>
                <h1>Hello, World!</h1>
            </body>
            </html>

    It was only the other day when I learnt about the "code" element and HTML "entities"! Kinda cool, hey?


  2. CSS-overall

This statement (at least for me) is simply untrue. I'm having a little bit of a hard time trying to grasp CSS, there seems to be so much to learn! Of course, with practice every day I will improve my skills!
